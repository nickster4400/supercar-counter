## Deploy to Vercel (FREE)

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Click "Import Project"
4. Select your GitHub repo
5. Click "Deploy"

Done! Your site will be live at `your-project.vercel.app`
